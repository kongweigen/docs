## 快速开始

### 1. 获取 API Key

在使用 API 之前，您需要先获取 API Key。请访问控制台获取您的专属密钥。

### 2. 发起第一个请求

以下是一个简单的对话请求示例：

```bash
curl -X POST "https://api.chatfire.cn/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "qwen2.5-72b-instruct",
    "messages": [
      {
        "role": "user",
        "content": "你好，请介绍一下自己"
      }
    ]
  }'
```

### 3. 处理响应

API 将返回 JSON 格式的响应，包含模型生成的内容和相关元数据。

## 认证方式

所有 API 请求都需要在请求头中包含有效的 API Key：

```
Authorization: Bearer YOUR_API_KEY
```

## 错误处理

API 使用标准的 HTTP 状态码来表示请求的成功或失败状态。详细的错误信息会在响应体中提供。

## 速率限制

为了确保服务的稳定性，我们对 API 请求实施了速率限制。具体限制取决于您的订阅计划。
