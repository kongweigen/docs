# Trigger Update

> Trigger an update after updating your OpenAPI document by calling this endpoint in a CI check

## OpenAPI

````yaml POST /project/update/{projectId}
paths:
  path: /project/update/{projectId}
  method: post
  servers:
    - url: https://api.mintlify.com/v1
  request:
    security:
      - title: bearerAuth
        parameters:
          query: {}
          header:
            Authorization:
              type: http
              scheme: bearer
          cookie: {}
    parameters:
      path:
        projectId:
          schema:
            - type: string
              required: true
              description: >-
                The ID of the project to trigger an update on. Can be retrieved
                from your dashboard.
      query: {}
      header: {}
      cookie: {}
    body: {}
  response:
    '202':
      application/json:
        schemaArray:
          - type: object
            properties:
              statusId:
                allOf:
                  - type: string
                    description: The status id of the triggered updated.
        examples:
          example:
            value:
              statusId: <string>
        description: A successful response
  deprecated: false
  type: path
components:
  schemas: {}

````
